============
House option
============

* Add a new area
* Add a new room in an area
* Add a widget at root of house or in an area or in a room

When you add a room it will first ask you where to put it.

When you add a widget, it will ask you witch widget, then where to put it (root,area,room) and display list of them. 

.. image:: image/fr_config_house.png

Don't forget to click on apply to save change.
